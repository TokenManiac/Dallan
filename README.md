# Dallan
A programming language which focuses on simplifying multithreaded &amp; parallelized work.

## Tasks:
1. Figuring out how to move data structures out of Rust, C++, C, C#, Java and proccess them.
2. Moving the processed data structures back to the source language in an idiomatic manner.
3. Implementing bytecode for debugging, LLVM for release builds.

Future: Implementing dallan in iot microcontrollers with multiple threads.

> Dallan in turkish means to "branch out".
